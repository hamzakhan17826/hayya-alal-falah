# Hayya Alal Falah - Islamic Books E-commerce

An e-commerce website for Islamic books, table calendars, Quran translations, and more. Built with Next.js, TypeScript, and Tailwind CSS.

## Features

- Modern, professional design with off-white and brownish/golden color scheme
- E-commerce functionality (add to cart, checkout)
- Direct book reading on the site
- Responsive layout
- TypeScript for type safety

## Getting Started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Run the development server:

   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `app/` - Next.js app router pages and layouts
- `components/` - Reusable UI components
- `public/` - Static assets

## Technologies Used

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS v4
- ESLint
